# TalentVerify AI
### Intelligent Candidate Screening and Professional Profile Verification System

## Project Overview

TalentVerify AI is an AI-powered web application designed to automate the resume screening process. The system allows candidates to upload resumes and enables recruiters to create job vacancies, evaluate applicants, and rank candidates based on AI-assisted resume analysis.

The project uses Google Gemini API to analyze resumes, compare them with job requirements, identify missing skills, and generate recommendations. Recruiters can use these insights to make faster and more consistent hiring decisions.

---

# Objectives

- Automate resume screening
- Reduce recruiter workload
- Match candidate skills with job requirements
- Generate AI-assisted candidate scores
- Provide resume improvement recommendations
- Rank candidates based on job compatibility
